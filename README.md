# How to run this
Clone the repository into your local machine:

`$ git clone https://github.com/hansnery/private-events.git`

Install the required gems:

`$ bundle install`

Run migration:

`$ rails db:migrate`

Run the server:

`$ rails server`

Access the web application locally:

`localhost:3000`
